Getting started

Installation

Regrid is available on npm and can be installed from the command line via npm or yarn: {% highlight console %} npm i regridjs --save {% endhighlight %} or {% highlight console %} yarn add regridjs {% endhighlight %}

Usage

  1. Import the package
    {% highlight js %} import Regrid from 'regridjs'; {% endhighlight %}
  2. Define an array with some data
    {% highlight js %} const data = [{ id: 1, firstName: 'John', lastName: 'Smith' }, ...]; {% endhighlight %}
